## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

## 相关依赖

``` bash
#  http://datav.jiaminghi.com/
npm install @jiaminghi/data-view
npm install echarts
npm install echarts-gl
npm install vue-router
```

## git步骤

``` bash
# 初始本地仓库
git init

# 查看当前状态 
git status 

# 看所有用户
git config --list

# 查看所有的分支 
git branch -a 

# 创建分支必须切换分支
git checkout -b main 

# 基于当前分支创建开发分支
git branch develop

```
